## 提示QQ版本过低
- 尝试更换协议
- 尝试终端输入: `pup fix --device ` 随机修改设备
## 提示网络异常
- 尝试切换同一网络(可使用代理)
- 尝试本地运行后，复制文件到服务器
- 尝试在 服务器 安装 官版QQ/Tim 进行登录，再运行PupBot
## 服务器繁忙
Server is busy
- 尝试更改网络DNS
- 119.29.29.29 / 114.114.114.114
## 群消息发送失败
- 账号被风控，需本地登录QQ，手动在群内发送信息，解除限制。
## 上网环境异常
- 使用扫码登录
- 尝试切换同一网络(可使用代理)
- 尝试在 服务器 安装 官版QQ/Tim 进行登录，再运行PupBot
## 提示密码已泄露
- 异地登录的原因，请打开设备锁。
